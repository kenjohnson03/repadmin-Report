  • Originating Site replicates to all domain controllers within a site and to a single bridgehead server in each subordinate site
  • Originating Server replicates to a subordinate servers
  • Destination Site replicates with the originating site based on NTDS site links in AD
  • Destination Server replicates with the originating servers based on NTDS settings seen in AD Sites & Services
  • Last Failure shows last date and time that an attempt was made to replicate with the Destination Server and was unsuccessful
  • Last Success shows last date and time a complete replication was performed
  • Status is a color indicator on how serious the problem is
